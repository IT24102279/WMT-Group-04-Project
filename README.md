# Pharmacy & Healthcare Management System

Monorepo containing:
- `backend/`: Node.js + Express + MongoDB API
- `mobile/`: Expo React Native app

## Quick Start

### Backend
1. Copy `backend/.env.example` to `backend/.env`.
2. Install deps: `cd backend && npm install`
3. Start server: `npm run dev`

### Mobile
1. Copy `mobile/.env.example` to `mobile/.env`.
2. Install deps: `cd mobile && npm install`
3. Run app: `npm run start`

## Modules
- Authentication
- Finance Tracking & Analytics
- Patient Tracking & Loyalty
- Inventory Management
- Customer Support & Appointments
- Online Shop & Delivery
- Sales Handling (POS)
